# Não Sei Onde Ir

## História dos Usuários

```
Achar um destino:
Como um turista
Eu gostaria de receber sugestões de lugares para passear
Para que eu podesse aproveitar o dia
```

```
Rotas de Ônibus:
Como um guia
Eu gostaria de saber as linhas de ônibus que passam no local
Para que eu possa chegar no local desejado
```

```
Custo:
Como numa bilheteria
Eu gostaria de saber o preço do lugar onde quero ir
Para que eu possa sair para um lugar que eu possa pagar
```

```
Horário:
Como um horário de funcionamento
Eu gostaria de saber o horário de funcionamento do local
Para que eu possa sair para um lugar que esteja aberto na hora que eu desejo ir
```

```
Localização:
Como num mapa
Eu gostaria de saber a localização do local
Para que eu possa sair pra um lugar onde eu possa chegar
```

```
Opiniões:
Como numa crítica
Eu gostaria de ler comentários sobre o lugar de pessoas que já visitaram
Para que eu possa ter algum feedback do lugar antes de sair de casa
```
